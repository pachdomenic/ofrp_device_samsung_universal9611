## Orange Fox device tree for the Samsung Exynos 9611 devices
Current targets:
- gta4xl
- m31

## How-to compile it:

To build:

```sh
. build/envsetup.sh
export ALLOW_MISSING_DEPENDENCIES=true
lunch twrp_gta4xl-eng
mka recoveryimage
```
